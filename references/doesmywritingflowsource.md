# Does My Writing Flow? - Paragraph Clarity Diagnostics

## Quick Flow Test

For any paragraph, ask:
1. Does it have ONE explicit message?
2. Does the FIRST sentence state what this paragraph will do?
3. Are all key nouns/terms readable without hidden context?
4. Does EACH sentence connect to the previous with a clear relation (cause, contrast, consequence, refinement, example)?

## Reverse Outlining

After writing a section, run this diagnostic:

1. **Write down the thesis** - What is the main claim of this section?
2. **Extract topic sentences** - Write the first sentence of each paragraph
3. **Extract evidence** - Under each topic sentence, list the evidence/explanation points
4. **Check mapping**:
   - Does each topic sentence directly support the thesis? If not, revise or remove.
   - Does the evidence under each paragraph support its topic sentence? If not, restructure.
5. **Check flow**:
   - Read just the topic sentences in order. Do they tell a coherent mini story?
   - If not, reorder paragraphs or add transition sentences.

## Flow Diagnostics by Paragraph Type

### Opening Paragraph
```
Signal: "In this paper, we address..."
Check: Does the reader know the problem AND the contribution within 3 sentences?
```

### Challenge Paragraph
```
Signal: "However, previous methods..."
Check: Is the technical reason for failure explicit? Not just "they perform poorly" but WHY.
```

### Method Paragraph
```
Signal: "To address this, we propose..."
Check: Motivation -> Design -> Advantage chain clear? Reader should know WHY before WHAT.
```

### Evidence Paragraph
```
Signal: "As shown in Table X..."
Check: Does the data support the claim without overclaiming? Are numbers contextualized?
```

## When Flow Feels Broken

1. **Add temporary section headers** - Break the draft into labeled subsections
2. **Add explicit transition phrases** during revision:
   - Cause: "This limitation arises because..."
   - Contrast: "In contrast to previous work,..."
   - Consequence: "As a result,..."
   - Refinement: "More specifically,..."
   - Example: "For instance,..."
3. **Remove unnecessary headers** before finalizing

## Red Flags

- Topic sentence appears at the END of a paragraph (burying the lead)
- Paragraph lacks a clear topic sentence entirely
- Two different ideas crammed into one paragraph
- Same idea spread across multiple paragraphs without clear progression
- Evidence presented without connection to the topic sentence
- Transitions are just "Furthermore..." or "Additionally..." without logical connection

## Quality Check

Can a reader who reads ONLY the first sentence of each paragraph understand the paper's argument?
